How to Use This Favicon

1. In Next.js

For Next.js 13+ with the App Router, update your app/layout.tsx file:

export const metadata: Metadata = {
  title: "Nico Herrera | Lead Software Engineer",
  description: "...",
  icons: {
    icon: [
      { url: '/favicon.svg', type: 'image/svg+xml' },
    ],
  },
};

2. For Better Browser Compatibility

For maximum compatibility, consider using a service like RealFaviconGenerator to convert the SVG to multiple formats.

3. Manual HTML Implementation

If you need to manually add it to your HTML, add this to your <head> section:

<link rel="icon" href="/favicon.svg" type="image/svg+xml">
